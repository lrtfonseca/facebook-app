Política de Privacidade

Última atualização: 2026

Esta política de privacidade descreve como as informações são tratadas pelo aplicativo utilizado para gerenciar publicações em páginas do Facebook.

Coleta de informações

O aplicativo não coleta, armazena ou compartilha informações pessoais de usuários do Facebook.

O aplicativo utiliza permissões da API do Facebook exclusivamente para publicar conteúdo em páginas administradas pelo próprio usuário que autorizou o acesso.

Uso das informações

As permissões concedidas são utilizadas apenas para:

  • Publicar vídeos ou posts em páginas do Facebook
  • Gerenciar conteúdo da página
  • Consultar informações básicas da página necessárias para a publicação

Nenhum dado é vendido, compartilhado ou utilizado para fins de marketing.

Armazenamento de dados

O aplicativo pode armazenar temporariamente tokens de acesso necessários para autenticação com a API do Facebook, exclusivamente para permitir a publicação automática de conteúdo.

Esses dados são utilizados apenas pelo administrador da página e não são compartilhados com terceiros.

Compartilhamento de dados

O aplicativo não compartilha dados de usuários com terceiros.
